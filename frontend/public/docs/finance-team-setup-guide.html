2. Getting Started

Follow these steps to create a new finance team user:

  1. Navigate to Roles - Go to Access Control ‚Üí Roles in the sidebar menu
  2. Click Finance Team Card - Select the green "Finance Team Roles" card at the top
  3. Click "Create Finance User" - Use the button in the top right corner
  4. Fill Basic Information - Enter first name, last name, email, and employee ID
  5. Select Role - Choose Accountant, Finance Controller, or Finance Auditor
  6. Configure Customizations - Set branch access, amount limits, and other restrictions
  7. Review & Save - Verify all settings and create the user
‚ö†Ô∏è Important: Finance roles have access to sensitive financial data. Always follow the principle of least privilege - only grant permissions that are necessary for the user's job function.

12. Best Practices

‚úÖ DO:
  • Follow the principle of least privilege
  • Regularly review and audit user permissions
  • Use customization options to limit access scope
  • Ensure proper separation of duties (entry vs approval)
  • Document reasons for any permission overrides
‚ö†Ô∏è DON'T:
  • Give Accountants approval authority without FC oversight
  • Allow Auditors to modify financial data
  • Share login credentials between users
  • Bypass the approval workflow for convenience
  • Leave inactive accounts with finance permissions
